亚马逊美食点评 背景¶

购物网络化带来的海量数据，商品性能评价，服务体验评价，原本口口相传的信息，借助互联网平台全部编程文本数据，一个经典的例子：根据twitter评论预测道琼斯股票指数的未来变化等，做情感分析对于提前预知消费者情绪的变化，以及作出相应的对策有十分重要的意义。

本篇文章使用的是Amazon 2012年食品区的商品评论数据（数据来源：kaggle）

##文本预处理函数定义

#定义转换小写函数
def clean_lower(text):
    text = text.lower()
    return text

#定义去除标点符号函数
clean_text = re.compile('[^a-z]+')
def clean_puc(text):
    text = clean_text.sub(' ',text).strip()
    return text

#提取词干
stemmer = SnowballStemmer('english')
def stem_text(text):
    text = stemmer.stem(text)
    return text
#定义分割+去除停止词
words=[]
def split_no_stop_word(text):
    for w in text.split(' '):
        if w not in stop:
            words.append(w)
    return words

#时间转换函数
def time_trans(text):
    return time.strftime('%Y',time.localtime(text))

#定义标签分类
def x_score(text):
    if text>3:
        return 1#positvie
    else:
        return 0#negative
